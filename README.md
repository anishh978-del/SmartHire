# 🚀 SmartHire – Swipe-Based Hiring Platform

SmartHire is a frontend prototype of a modern hiring platform that connects students and companies using a swipe-based interaction model.

---

## ✨ Features

### 👨‍🎓 Student Side
- Swipe-based job discovery (like Tinder)
- Match percentage with explanation
- Resume preview system
- Dashboard with analytics (Chart.js)
- Animated stats (profile views, applications, matches)

### 🏢 Company Side
- Swipe through candidates
- Match-based filtering
- Resume preview popup
- Company dashboard with hiring analytics
- Interactive charts and stats

---

## 📊 Dashboard Highlights
- Interactive graphs using Chart.js
- Animated counters (profile views, applications, matches)
- Role-based dashboards (Student & Company)

---

## 🧠 Concept

The goal of SmartHire is to:
- Reduce irrelevant job applications  
- Improve matching accuracy  
- Make hiring faster and more engaging  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Chart.js  

---

## 🚀 Future Improvements
